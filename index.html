Projects

Paranormal Places

Airbnb inspired me to create a project that put a twist on renting in a spooky way. I'm currently working on Paranormal Places, a haunted locations bookings site. I created this app using MEAN stack (MongoDB, Express, Angular, Node.js). It's a project I'm ambitious to improve, as I am currently studying the best way to set up the database. I especially love the playful animated ghost because it offers a memorable user experience. Plus, it's been really fun to make! See my work and read about my progress on github.

WDI Eat

For a group project, I was a contributor to WDI Eat (what do I eat?). It's an app that helps users make a very important life decision: What they eat for their next meal! This app uses the Yelp API to return a random restaurant based on paramaters set by the user. The app can also provide a completely random result if no parameters are chosen. In addition to Yelp, we used Google Maps API on the results page to show the location and directions to the eatery. Another goal for this project was to implement CRUD actions, create an API, and login/logout through our database as well as facebook. I had fun creating the background images and making them change when you refresh. Check out the website here!
